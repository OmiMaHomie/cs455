# EXECUTION STEPS
    The files are already coded out to be run on their own. To run the files, do the following:
    1. In one linux terminal, initiate the server code.
    - ~~~$ python3 server.py <port> (any port from 58000-58999)
    - EXAMPLE:  python3 server.py 58805

    2. In another linux terminal, run the client file.
    - ~~~$ python3 client.py <hostname> <port> (same port restrictions, 58000-58999)
    - EXAMPLE:  python3 client.py localhost 58005 
        OR
        python3 client.py csa1.bu.edu 58005'

    3. Look at both terminals.
    - Server should read something like:
        Server listening on port XXXXX...
        Connection from ('XXX.X.X.X', XXXXX)
        Received: ~~~~~~~~~~.
        Sent: ~~~~~~~~.
        Connection closed
    - Client should be something like:
        Connected to XXXXXXXX:YYYYY
        Sent: ~~~~~~~~~.
        Received: ~~~~~~~~~.
